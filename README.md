\# BDC Project



\## Introduction

This is a Brushless DC Motor control program based on the \*\*HC32\*\* microcontroller.



\## Development Environment

\- IDE: Keil uVision 5 / IAR EWARM

\- Compiler: Arm Compiler V6

\- Hardware: BDC Control Board V1.0



\## How to Sync

Run `./sync.sh` to push changes to GitHub and Gitee simultaneously.
